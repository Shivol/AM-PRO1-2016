# Лабораторная работа №15
### Динамические структуры данных. Двусвязные Списки

1. Посчитать количество нулевых элементов в списке.
2. Описать функцию проверки списка на симметричность.
3. Вставить новое значение в середину списка. Или, если число элементов нечётное, перед центральным элементом списка. 
4. Удалить заданное количество элементов, перед первым элементом с заданным значением. 
5. Вставить заданное значение после каждого элемента с другим заданным значением.
6. Разделить список на два списка, оставив в одном списке элементы с чётными номерами, а в другом с нечётными.
7. _Дополнительное задание._
  Описать функцию сортировки списка алгоритмом быстрой сортировки.
