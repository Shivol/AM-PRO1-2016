# Лабораторная работа №6
### Динамические массивы

Все задачи решаются с использованием динамических массивов. Каждое задание оформляется в виде функции обработки массива. Обработка ошибок должна осуществляться посредством исключений. Следует использовать [метод контроля утечек памяти](http://msdn.microsoft.com/ru-ru/library/x98tx3cf.aspx). Для каждого задания следует написать тесты.

 0. Дан массив целых чисел. Продублировать в массиве все элементы, находящиеся между первым и последним нулевыми элементами. Если вставка невозможна выдать соответствующее сообщение.

 1. Дан массив вещественных чисел. Удалить из массива все элементы, меньшие среднего арифметического элементов массива. 

 2. Даны массивы целых чисел. Продублировать в массиве все элементы, находящиеся между парами нулевых элементов. Если вставка невозможна выдать соответствующее сообщение.

 3. Даны два массива целых чисел. Удалить из первого массива все вхождения второго массива.

 4. Дан массив вещественных чисел. Вставить между каждой парой элементов значение разности этих элементов.

 5._Дополнительное задание._ Дан массив целых чисел. Реализовать алгоритм [сортировки слиянием](https://ru.wikipedia.org/wiki/%D0%A1%D0%BE%D1%80%D1%82%D0%B8%D1%80%D0%BE%D0%B2%D0%BA%D0%B0_%D1%81%D0%BB%D0%B8%D1%8F%D0%BD%D0%B8%D0%B5%D0%BC) с использованием динамической памти.
