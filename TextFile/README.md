# Лабораторная работа №10
### Текстовые файлы 

1. Дан текстовый файл, состоящий из строк. Вывести файл построчно, указав для каждой строки её длину. 

2. Дан текстовый файл, состоящий из строк. Посчитать, сколько раз в файле встречается каждая буква латинского алфавита.

3. Дан текстовый файл, содержащий матрицу целых чисел. Создать новый файл, содержащий транспонированную матрицу в том же формате. 
 Исходный файл имеет следующий формат:
```C++
    n m
    a11 a12 … a1m
    …
    an1 an2 … anm
```

5. Дан файл содержащий строки вещественных чисел. Вычислить минимум и максимум в каждой строке. Результат записать в файл.

6. Дан текстовый файл, содержащий строки слов. Выделить все вхождения заданного слова, заменив его буквы на заглавные.
