#  Лабораторная работа №11
### Структуры. Двоичные файлы

1. Дан двоичный файл, содержащий данные о точках типа `point`:
 * Посчитать количество точек в файле.
 * Найти точки, которые удовлетворяют условию `y = x^2`.
 * Среди всех троек точек, найти треугольник с максимальной площадью
  (используя функции из Лабораторной работы №1).

2. Создать двоичный файл, который содержит данные о человеке, используя структуру `person` :
 * Прочитать из файла сначала данные лиц мужского пола, затем женского.
 * Найти в файле всех людей с заданной фамилией.
 * _Дополнительное задание._
  Создать файл, содержащий данные лиц, чей возраст меньше заданного числа _N_ (используя библиотеку [`<ctime>`](http://www.cplusplus.com/reference/ctime/mktime/)).
