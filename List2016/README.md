# Лабораторная работа №13
### Динамические структуры данных. Списки

0. Посчитать количество нулевых элементов в списке.
1. Удалить из списка все значения равные заданному.
2. Вставить новое значение в середину списка. Или, если число элементов нечётное, перед центральным элементом списка. 
3. Заполнить список значениями из файла, расположив их в порядке не убывания значений.
4. Удалить заданное количество элементов, после первого элемента с заданным значением. 
5. Вставить заданное значение пред каждым элементом с другим заданным значением.
6. Разделить список на два списка, оставив в одном списке элементы с четными номерами, а в другом с нечетными.
