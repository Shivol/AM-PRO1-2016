# Лабораторная работа №8
### С-cтроки

При решении данных заданий, использовать строки статического размера.

К каждому заданию необходимо написать тесты. 
#### Реализация функций cstring
При решении следующих задач нельзя использовать стандартные функции обработки строк.

 1. Сравнить две строки на равенство ([strcmp](http://cppstudio.com/spravochnik/standartnye-zagolovochnye-fajly-iz-si-v-s/zagolovochnyj-fajl-cstring-string-h/funkciya-strncmp/)).
 2. Найти в строке первое вхождение заданного символа ([strchr](http://cppstudio.com/spravochnik/standartnye-zagolovochnye-fajly-iz-si-v-s/zagolovochnyj-fajl-cstring-string-h/funkciya-strchr/)).
 3. Удалить из строки каждое вхождение заданного символа.
 4. Вставить в строку другую строку после первого вхождения заданного символа. Если это невозможно, выдать сообщение об этом.

#### Обработка слов
Дана строка, состоящая из слов, разделенных одним или несколькими пробелами.
При решении этих задач можно использовать [стандартные функции](http://cppstudio.com/cat/309/325/) обработки строк.

 1. Посчитать количество слов в строке (с кириллицей).
 2. Найти в строке заданное слово.
 3. Заменить в строке указанное слово, на другое слово.
 4. _Дополнительное задание._ 
   Отсортировать слова в строке в лексикографическом порядке.
