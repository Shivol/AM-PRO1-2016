# Лабораторная работа №9
### Динамические строки. Обработка строк

#### C-string
При решении следующих заданий следует использовать строки `char*`, размещаемые в динамической памяти и стандартные функции обработки строк.
	
1. Дана строка, поменять местами подстроки до и после вхождения заданного символа. Вхождение символа считать единственным.

2. Дана строка, удалить из строки все вхождения заданной подстроки.

3. Дана строка, состоящая из слов, разделенных одним пробелом. Получить строку содержащую те же слова в обратном порядке.

#### str::string
При решении следующих задач следует использовать строки `std::string`.

4. Дана строка, удалить из строки все вхождения заданного символа.

5. Дана строка, состоящая из слов, разделенных одним или несколькими пробелами. Заменить в строке каждое четное слово на заданное.

6. Дана строка, состоящая из слов, разделенных одним или несколькими пробелами. 
 Создать массив строк, элементами которого являются слова исходной строки. Вычислить количество вхождений каждого из слов в исходную строку.
