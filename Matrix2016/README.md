# Лабораторная работа №7
### Двумерные массивы

Каждую задачу необходимо решить дважды: с использованием статических и динамических массивов. 
Описать тесты для каждой функции. При использовании динамической памяти необходимо контролировать утечки памяти.

 1. Сравнить две матрицы на равенство.

 2. Удалить нулевые столбцы из матрицы.

 3. Добавить в матрицу строку с номером _К_. Вектор-строка передается как параметр.

 4. Транспонировать не квадратную матрицу.

 5. Найти произведение двух не квадратных  матриц.

 6. _Дополнительное задение._  
  Дана квадратная матрица вещественных чисел. 
  Вычислить определитель матрицы, пользуясь [формулой разложения по первой строке](https://ru.wikipedia.org/wiki/%D0%9E%D0%BF%D1%80%D0%B5%D0%B4%D0%B5%D0%BB%D0%B8%D1%82%D0%B5%D0%BB%D1%8C#.D0.9C.D0.B0.D1.82.D1.80.D0.B8.D1.86.D1.8B_N_x_N). 
  Использовать динамическое выделение памяти.
