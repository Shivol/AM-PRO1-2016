#Лабораторная работа №12
### Структуры в динамической памяти

Используя структуру `student`:
1. Создать двоичный файл из данных текстового файла. 
2. Удалить из двоичного файла данные о студентах с нулевым суммарным баллом, не создавая нового файла.
3. Выдать список студентов, отсортированный по суммарному баллу. 
4. Разбить данные на два файла по номеру группы, отсортировав студентов по баллу. 
5. Объединить данные из двух отсортированных файлов в один сохраняя порядок сортировки.
